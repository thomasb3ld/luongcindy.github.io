Progress

We ordered the materials and started to parse through the source code of the ChessAI. We also planned the electrical component design with reed sensors and multiplexers. Finally, we planned the physical design with wood and laser cutting

Weekly Objectives

We wanted to first build the physical chess board where we were able to complete within the first weeks. We also started to learn how to code on the Arduino with the FastLED library. Finally, we worked on the Reed Sensor grid and testing to see if they worked.

Challenges

We struggled in the beginning with testing the multiplexers. Especially, because we didn't know if the multiplexers were bugged or if we were testing them incorrectly. In terms of software, we didn't know if the code worked. Finally, we had issues with choosing the wrong dimensions and had to recalculate.

Electrical Progress

We were able to solder all of the ribbon cables to the multiplexers and the reed sensors and also soldered the wires to the other end of sensors for input voltages. We tested all of the sensores, LEDs, and Arduino

Software Progress

We were able to understand the source code for the AI and how it evaluates the board. We also marked segments of the code that weren't necessary in our implementation and understood how to use the LED code.
