Electrical Flow
Player moves chesspiece with a magnetic bottom onto a square.
A reed sensor underneath the square senses the magnet's field and sends a 5 volt signal through a connected wire.
The wire carries the 5 volt signal to one of the multiplexers, which is in turn connected to the Arduino.
The Arduino recieves the signal, and the code within calculates which square the player should place their piece next. The code outputs the LED index of this new square.
The new index information is sent out of the Arduino through output pins, which are connected to wires. These wires transfer the output to the LEDs.
The LED at the new square lights up green! If the player tries to move their chesspiece to a different square, the lit up square will shine red until the chesspiece is moved back.
Materials List
  • Reed Sensors
  • Count: 64
  • Soldered underneath the chessboard squares and to wires that connect to multiplexer(s)
  • Sends signal current to multiplexer(s) if magnetic field from chesspieces is sensed
  • 16-to-1 Multiplexers
  • Count: 4
  • Bridge between electrical hardware and software
  • Each one takes input from 2 rows of the board
  • Arduino Uno REV3
  • DC 5521 input for direct powerbank connection
  • covered by a Terminal Block Breakout Module to decrease precision soldering
  • 12 Volt Power Bank
  • LED Strips
